What is Tor really used for?

Tor's usage may vary on how the user chooses to interact with tor. HOWEVER, what is Tor ACTUALLY used for? It may be quite saddening to hear that not only Tor can be used to conceal one's identity, but can also be used to access a certain part of the internet which isn't visible to regular search engines and requires the use of tor and other anonymizing softwares.
This is called the deep-web.

Therefore, the following list may be a checklist for one's safety before surfing through deep-web out of curiosity.
  • Are you sure you are connected to the Tor network? (Visit the connection-check website to check)
  • Have you enabled the default safety features? (e.g Noscript&Https-Everywhere)
  • Have you made sure that you did not resize the window? (This is for security reasons)
  • Have you made sure that there are no other tabs open apart from Tor? (Again, this is for security reasons)
If the criteria are met, the user should be set to browse on the deep-web. The video below is a good beginner's tutorial. Tor.

Useful Onionlinks!

Unlike usual domain names users are familiar to, e.g: .com,.net,.org, the only domain names users would find in the deep-web are .onion.
This is the special-use TLD(Top Level Domain) suffix designating an anonymous onion service (formerly known as a "hidden service") reachable via the Tor network.
Because .Onion links are at most times, very long, it may be hard for users to find websites. So, the table below will list some popular websites in .Onion domain names. (Malicious websites excluded from list)
